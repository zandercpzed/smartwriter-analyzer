# Development Notes

Anotações e notas técnicas durante o desenvolvimento.

## Organização

Use a nomenclatura:
- `YYYY-MM-DD - Descrição da tarefa.md`

Exemplo:
- `2026-01-13 - Refinar AI Detection.md`
- `2026-01-15 - Adicionar testes para readability.md`

## Processo

1. Criar nota quando começar uma tarefa
2. Adicionar contexto, decisões, problemas encontrados
3. Linkar arquivos e commits relevantes
4. Atualizar quando a tarefa for completada
