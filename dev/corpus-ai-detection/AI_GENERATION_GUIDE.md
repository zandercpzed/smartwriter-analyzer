## ðŸ“ Step-by-Step Process

### 1. Get the Prompts
```bash
cat dev/corpus-ai-detection/PROMPTS.md
```

You'll see prompts numbered and organized by provider:
- Prompts #1-8: For ChatGPT 3.5
- Prompts #1-4: For ChatGPT 4
- Prompts #1-4: For Claude
- Prompts #1-2: For Gemini
- Prompts #1-2: For Meta

### 2. Generate Text

**For ChatGPT 3.5 - First Text**:
1. Open chatgpt.com
2. Find Prompt #1 in PROMPTS.md (ChatGPT 3.5 section)
3. Copy entire prompt exactly
4. Paste into ChatGPT
5. Send and wait for response
6. Copy complete response

### 3. Save with Template Format

Follow `TEMPLATE_GENERATED.md`:

```markdown
# AI Generated Text - [Model]

## Metadata
- **Generated By**: [AI Model Name]
- **Prompt Source**: Prompt #[X] from PROMPTS.md
- **Generation Date**: [Today's Date]
- **Word Count**: [Count words]
- **Language**: Portuguese (Brazil)
- **Label for Training**: GENERATED

## Prompt Used
[Copy the prompt used from PROMPTS.md here]

## Generated Text
[Paste the full AI-generated response here]

## Metadata Notes
- Provider: [ChatGPT 3.5 / ChatGPT 4 / Claude / Gemini / Meta]
- Model Version: [e.g., GPT-3.5-turbo, Claude 3 Sonnet, etc.]
- Temperature Setting: [If applicable]
- Length: [X words]
- Generated: [Date and time]
```

### 4. Save File with Correct Name

**Location**: `dev/corpus-ai-detection/generated/[provider]/[number].md`

Examples:
- `dev/corpus-ai-detection/generated/chatgpt-3.5/001.md`
- `dev/corpus-ai-detection/generated/chatgpt-4/001.md`
- `dev/corpus-ai-detection/generated/claude/001.md`
- `dev/corpus-ai-detection/generated/gemini/001.md`
- `dev/corpus-ai-detection/generated/meta/001.md`

### 5. Update metadata.csv

Add entry to `dev/corpus-ai-detection/metadata.csv`:

```csv
generated/chatgpt-3.5/001.md,ChatGPT 3.5,GENERATED,ChatGPT 3.5,WORD_COUNT,pt-BR,2026-01-13,Generated from Prompt #1
```

## ðŸ” Quality Notes

**What Makes a Good AI Text for This Corpus**:
- Generated in Portuguese (Brazil) or translatable
- Response is natural and coherent
- Length: 300-500 words (or whatever the AI generates naturally)
- No obvious repetition or errors (unless part of AI's actual output)
- Clear that it came from an LLM (for labeled training data)

**What to Avoid**:
- Partial responses (always get full output)
- Multiple attempts for same prompt (use first generation)
- Heavily edited or manually corrected text (use as-generated)
- Off-topic or error responses (regenerate if completely off)
