## Text

Comprei esse notebook para uso profissional há quatro meses, com expectativas moderadas pelo preço. Devo dizer que foi uma grata surpresa em quase tudo.

A bateria é a característica mais impressionante. Afirmam 8 horas, e conseguem 7-7.5 em uso real, que é próximo ao anunciado. Comparado com meu notebook anterior, é noite e dia. Antes, eu estava constantemente buscando uma tomada. Agora, posso sair de casa, passar o dia inteiro fora, e ainda ter bateria para trabalhar à noite.

O teclado é confortável para digitação prolongada. Não é mecanicamente perfeito, mas para um notebook fino, é surpreendentemente responsivo. Após quatro meses digitando horas todos os dias, não sinto fadiga nas mãos como sentia com outros modelos.

Processamento é adequado para minhas necessidades. Não estou jogando games ou fazendo renderização 3D, mas para desenvolvimento web, edição de documentos e múltiplas abas do navegador, o processador é mais que suficiente. Às vezes sinto que está trabalhando (fãs ligam), mas nunca parou de responder.

Meus problemas:

1. A webcam é péssima. Reuniões por vídeo com essa câmera é deprimidor. Investi em uma webcam externa, que deveria estar incluída nesse preço.

2. O trackpad é pequeno. Para alguém vindo de um MacBook, é uma adaptação desconfortável. Usar mouse externo resolve, mas isso é uma limitação real para uso portátil.

3. As portas são adequadas, mas limitadas. Apenas dois USB-C e um USB-A. Para alguém que precisa conectar múltiplos periféricos, é apertado.

4. Esquenta um pouco na base durante uso intenso, embora nunca questionavelmente quente.

No geral, para o preço, é excelente. Se você procura um notebook confiável, com boa bateria e processamento suficiente, sem pagar o premium das marcas luxo, vale considerar. Apenas não compre esperando recursos premium - você vai se decepcionar. Compre esperando bom custo-benefício, e será surpreso positivamente.
