# Desenvolvimento — SmartWriter Analyzer

Área de desenvolvimento organizada para facilitar o trabalho na Fase 1 e além.

## Estrutura

- **development-notes/** — Anotações e notas técnicas de desenvolvimento
- **test-manuscripts/** — Manuscritos de teste para validação de análises
- **analysis-results/** — Resultados de análises para comparação e validação

## Workflow

1. Criar nota em `development-notes/` para cada tarefa
2. Testar com manuscritos em `test-manuscripts/`
3. Salvar resultados em `analysis-results/` para rastreabilidade
